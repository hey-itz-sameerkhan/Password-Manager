# ✨ Overview

**SecurePass Password Manager** is a simple yet powerful web application that helps users **generate, manage, and organize passwords securely**.

The application includes **password generation tools, strength indicators, dark mode UI, and clipboard utilities** to improve productivity and security awareness.

# 🧠 Project Philosophy

This project focuses on three main ideas:

🔐 **Security Awareness**  
Encourages users to generate stronger passwords.

⚡ **Simplicity**  
Clean and minimal UI for quick usage.

🎨 **User Experience**  
Modern design with smooth interactions.

> A secure web experience begins with strong passwords.

# 🧩 Tech Stack

| Layer | Technologies | Purpose |
|------|--------------|--------|
| Structure | HTML5 | Semantic page structure |
| Styling | CSS3, Flexbox | Responsive UI design |
| Logic | JavaScript (Vanilla) | Password generation and UI interactions |
| UI Features | DOM Manipulation | Dynamic password management |
| Deployment | Netlify | Hosting platform |

---

# 💼 Core Features

| Feature | Description |
|-------|-------------|
| 🔑 Password Generator | Generate passwords with different strength levels |
| 📊 Password Strength Indicator | Visual indicator showing password security |
| 👁 Show / Hide Password | Toggle password visibility |
| 🌙 Dark Mode | Switch between light and dark UI |
| 📋 Copy Password | Copy passwords instantly to clipboard |
| ❌ Delete Password | Remove saved passwords easily |
| 📱 Responsive UI | Works on mobile, tablet, and desktop |
